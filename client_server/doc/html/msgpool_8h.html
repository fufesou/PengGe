const char *  data,
int  datalen

cspool_pushdata This function push one buffer item data to pool. This procedure can be splitted into two steps:

  1. take out one item from the empty buffer.
  2. copy data from 'data' to the buffer item.
  3. push buffer item into the filled buffer.
Parameters
poolthe operating pool.
data
datalen
Returns
0 if success. There is at least one empty buffer. 1 if fail. There is no empty buffer. -1 if fail. copy data error.
