Member Data Documentation

int csmsgpool::threadexit

threadexit threadexit is used to make thread exit normally. If threadexit is 1, thread will exit; if threadexit is 0, thread keep on running.

Note
thread entry must test this value to determine whether exit or not, as the following code shows.
func()
{
while (1) {
if (threadexit) {
break;
}
}
}
int csmsgpool::use_sem_in_pool

use_sem_in_pool 1 turn on semaphore wait or post, 0 turn off.

Note
use_sem_in_pool only works in cspool_pulldata and cspool_pushdata. user shoud call semaphore wait and post explicit while calling cspool_pullitem and cspool_pushitem.
